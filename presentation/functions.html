üìö Libraries & Dependencies

<GL/glut.h>

The OpenGL Utility Toolkit. This is the core framework used for the project.

  • Windowing: Handles creating the window and the main event loop (`glutMainLoop`).
  • Input: Manages keyboard (`glutKeyboardFunc`) and mouse interactions.
  • Primitives: Provides built-in shapes like `glutSolidSphere`, `glutSolidCube`, and `glutSolidCone` which are used as base geometry.
  • Note: Automatically includes `gl.h` (Core OpenGL) and `glu.h` (GL Utilities).
<math.h>

Standard C Math Library. Essential for the procedural generation and animation logic.

  • Trigonometry: `sin()` and `cos()` are used for the ocean waves, ship rocking motion, and circular geometry (curved stern).
  • Camera: Used to calculate the camera's position when orbiting or following the ship.
<stdlib.h>

Standard General Utilities Library.

  • Randomization: Uses `rand()` to create variations in particle effects (wake bubbles) so they don't look artificial.

Renders a single rock using a scaled glutSolidDodecahedron. Used in loops to create rocky shorelines.

üö¢ Ship Components

void drawShipModel()

The master function that assembles the entire ship by calling sub-component functions (hull, deck, bridge, propellers) in a hierarchical manner.

void drawCurvedStern()

Generates the rounded back of the ship using GL_TRIANGLE_FAN and trigonometric calculations to create a smooth curve.

void drawBridge()

Constructs the Captain's bridge area using stacked scaled cubes and window cutouts.

void drawPropellerAssembly()

Draws the complex propeller mechanism including V-struts, shaft, and rotating blades.

void drawDeckWindows(float length, float width, float height)

Procedurally generates a row of windows along the specified deck dimensions.

üåç Environment & Effects

void drawMountains()

Renders the three main islands by combining multiple `drawPyramid` calls with trees and rocks placed at specific coordinates.

void drawOcean()

Creates the dynamic water surface using a high-density vertex grid animated with Sine and Cosine waves based on time.

void drawCoconutTree()

Constructs a palm tree using a cylinder for the trunk and rotated cones for the foliage leaves.
